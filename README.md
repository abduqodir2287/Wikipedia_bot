# Wikipedia Bot

**Wikipedia Bot** – это Telegram-бот, который предоставляет информацию из Википедии в формате, удобном для чтения, на выбранном языке.

## Описание

Проект **Wikipedia Bot** позволяет пользователям взаимодействовать с Википедией через Telegram, выполняя следующие действия:
- **Приветствие**:
  - После команды `/start` бот приветствует пользователя и предлагает выбрать один из трёх языков.
- **Поиск информации**:
  - Пользователь вводит запрос, и бот возвращает ответ из Википедии на выбранном языке в красивом формате.
- **Смена языка**:
  - В любой момент можно сменить язык общения через команду `/change_language`.
- **Помощь**:
  - Команда `/help` предоставляет информацию о доступных функциях бота.

## Основные функции

- **Поддержка нескольких языков**:
  - Бот позволяет выбирать между тремя различными языками.
- **Запросы к Википедии**:
  - Быстрое получение информации с использованием API Википедии.
- **Команды**:
  - `/start`: запуск бота и выбор языка.
  - `/help`: информация о командах бота.
  - `/change_language`: смена языка общения с ботом.

## Технологии

Проект использует следующие технологии и библиотеки:
- **Aiogram 3.13.1**: для взаимодействия с Telegram API.
- **Pydantic**: для валидации данных.
- **Wikipedia**: для получения информации из Википедии.

## Установка

1. **Клонируйте репозиторий**:

    ```sh
    git clone https://github.com/abduqodir2287/Wikipedia_bot
    cd Wikipedia_bot
    ```

2. **Создайте и активируйте виртуальное окружение**:

    ```sh
    python -m venv venv
    source venv/bin/activate  # Для Windows используйте `venv\Scripts\activate`
    ```

3. **Установите зависимости**:

    ```sh
    pip install -r requirements.txt
    ```

4. **Настройте переменные окружения**:

   Скопируйте файл `.env_example` в `.env` и заполните его своими данными:

    ```sh
    cp .env_example .env
    ```

## Использование

1. **Запустите бота**:

    Используйте команду `make start-bot`, чтобы запустить бота:

    ```bash
    make start-bot
    ```

   Либо без `make`:

    ```bash
    python -m src.main
    ```

## Docker

Для удобного развертывания проекта можно использовать Docker.

1. **Запустите проект с помощью Docker**:

    ```bash
    docker-compose up --build
    ```

## Примеры использования

- **Начало работы**:
  Введите команду `/start`, выберите язык и отправьте запрос для поиска информации.
- **Изменение языка**:
  Используйте команду `/change_language`, чтобы сменить язык.
